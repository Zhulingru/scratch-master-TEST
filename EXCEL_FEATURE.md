# Excel 導出功能說明

## 📊 功能介紹

現在你的測驗應用可以將測驗結果導出為 Excel 檔案！

## 🎯 功能特點

### 導出的 Excel 檔案包含兩個工作表：

#### 1. **測驗摘要** 工作表
- 測驗日期
- 總分（例如：80 / 100）
- 答對題數（例如：8 / 10）
- 正確率（例如：80.0%）

#### 2. **詳細答案** 工作表
包含每題的詳細資訊：
- 題號
- 題目內容
- 你的答案
- 正確答案
- 是否正確（✓ 或 ✗）
- 題目類別

## 🚀 使用方法

1. **完成測驗**：回答所有 10 道題目
2. **提交答案**：點擊「送出答案」按鈕
3. **查看結果**：系統會顯示你的分數
4. **導出 Excel**：點擊「導出 Excel」按鈕
5. **下載檔案**：瀏覽器會自動下載 Excel 檔案

## 📁 檔案格式

- **檔案名稱**：`Scratch測驗結果_YYYY-MM-DD.xlsx`
- **格式**：Excel (.xlsx)
- **編碼**：UTF-8（支援中文）

## 💡 使用場景

- ✅ **學生記錄**：保存測驗結果，追蹤學習進度
- ✅ **老師分析**：收集學生答案，分析錯誤模式
- ✅ **成績管理**：匯入到其他系統進行統計
- ✅ **備份資料**：保存測驗歷史記錄

## 🔧 技術實現

### 使用的套件
- **xlsx**：處理 Excel 檔案的 JavaScript 函式庫

### 檔案結構
```
utils/
  └── excelExport.ts  # Excel 導出功能
App.tsx               # 添加了導出按鈕
```

### 程式碼範例

```typescript
// 導出功能
exportToExcel({
  questions: questions,        // 題目列表
  userAnswers: userAnswers,    // 用戶答案
  score: score,                // 分數
  totalQuestions: 10           // 總題數
});
```

## 📝 注意事項

1. **瀏覽器相容性**：現代瀏覽器都支援（Chrome、Firefox、Safari、Edge）
2. **檔案大小**：Excel 檔案通常很小（< 100KB）
3. **中文支援**：完全支援繁體中文
4. **下載位置**：檔案會下載到瀏覽器的預設下載資料夾

## 🎨 UI 設計

導出按鈕位於測驗結果頁面，與「再測驗一次」按鈕並排顯示：
- 白色背景，藍色文字
- 帶有下載圖示
- 懸停時有放大效果
